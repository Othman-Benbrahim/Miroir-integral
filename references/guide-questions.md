# Guide de Questions — Miroir Intégral

> Banque complète de questions par couche et par signal.
> Utiliser pendant l'ÉTAPE 1b pour choisir la prochaine question
> en fonction de ce qui émerge dans les réponses.
> **Ne jamais poser plus d'une question à la fois.**

---

## Règles de sélection

1. **Suivre le signal le plus chargé** — si une réponse contient une émotion
   forte, une contradiction ou une hésitation, creuser là avant d'avancer.
2. **Alterner les niveaux** — ne pas rester 5 questions sur la même couche.
3. **Couche 4 (Ombre)** : uniquement en MODE COMPLET, après au moins 5 autres
   questions, quand un minimum de confiance est établi.
4. **Maximum 12 questions** — arrêter et construire le bilan même si tout n'a
   pas été exploré. Un bilan partiel et honnête vaut mieux qu'une enquête épuisante.

## Banque par couche

### COUCHE 1 — EMS et Kegan

**Pour détecter les schémas actifs (EMS) :**

| Question | Signal recherché |
|---|---|
| "Qu'est-ce que vous vous dites à vous-même dans ces moments ?" | Voix intérieure critique, injonctions automatiques |
| "Cette conviction — depuis quand est-elle là ?" | Origine du schéma (enfance/adolescence) |
| "Qu'est-ce qui se passerait de pire si vous changiez ça ?" | Peur centrale du schéma |
| "Est-ce que vous connaissez quelqu'un qui vit ce que vous voulez — comment ils font ?" | Comparaison, envie, sentiment d'incapacité |
| "Si vous ne deviez pas réussir pour le prouver à quelqu'un — que feriez-vous ?" | Standards élevés, orienté vers l'autre |
| "Qu'est-ce qui se passe dans votre corps quand vous pensez à faire ce changement ?" | Ancrage somatique du schéma |

**Pour détecter la position Kegan :**

| Question | Signal Kegan 3 | Signal transition / Kegan 4 |
|---|---|---|
| "Qu'est-ce qui compte vraiment pour vous, indépendamment du regard des autres ?" | Difficulté à répondre sans référence externe | Réponse claire et propre |
| "Y a-t-il des choses que vous faites uniquement parce que vous vous sentez obligé ?" | Oui, beaucoup, difficile d'en sortir | Peut les nommer et questionner |
| "Qu'est-ce qui se passe en vous quand vous déçevez quelqu'un d'important ?" | Effondrement, honte intense, besoin de réparer | Inconfort gérable, position maintenue |
| "Qui attendez-vous que vous soyez dans cette situation ?" | Réponse centrée sur les attentes des autres | "Moi-même" ou tension consciente entre les deux |

### COUCHE 2 — IFS (Parties internes)

**Pour identifier les Managers :**

| Question | Ce qu'elle révèle |
|---|---|
| "La partie de vous qui freine — qu'est-ce qu'elle essaie d'éviter ?" | Intention positive du Manager |
| "Si vous laissiez tomber ce contrôle — qu'est-ce qui se passerait de pire ?" | Ce que le Manager protège |
| "Cette voix intérieure qui vous critique — elle ressemble à qui ?" | Origine (parentale, scolaire, culturelle) |
| "Qu'est-ce que cette partie de vous a dû gérer, à une époque ?" | Histoire du Manager |

**Pour identifier les Exilés :**

| Question | Ce qu'elle révèle |
|---|---|
| "Quand cette émotion monte — à quel âge vous sentiez-vous déjà comme ça ?" | Age de l'Exilé |
| "Y a-t-il une image ou un souvenir qui vient avec cette sensation ?" | Mémoire de l'Exilé |
| "Qu'est-ce que cette partie de vous aurait eu besoin d'entendre, à l'époque ?" | Besoin non satisfait |

**Pour accéder au Self :**

| Question | Ce qu'elle révèle |
|---|---|
| "Y a-t-il une partie de vous qui peut observer tout ça avec un peu de distance ?" | Présence du Self |
| "Quand vous êtes vraiment calme — comment voyez-vous cette situation ?" | Qualité du Self quand disponible |
| "Si vous pouviez parler à cette partie de vous avec bienveillance — que lui diriez-vous ?" | Capacité de Self-leadership |

### COUCHE 3 — Récit (McAdams)

| Question | Ce qu'elle révèle |
|---|---|
| "Si votre histoire avait un titre — quel serait-il ?" | Séquence dominante, tonalité |
| "Racontez une situation passée où tout allait bien, et comment ça a basculé." | Séquence contamination |
| "Y a-t-il quelque chose de difficile qui vous a finalement appris quelque chose d'important ?" | Séquence rédemption |
| "Y a-t-il quelqu'un — réel ou imaginaire — dont la voix résonne quand vous vous critiquez ?" | Imago |
| "Quel tournant dans votre histoire aurait pu tout changer, selon vous ?" | Tournant narratif disponible |
| "Si cette histoire devait avoir une suite différente — qu'est-ce qui devrait se passer ?" | Arc de transformation disponible |

---

### COUCHE 4 — Ombre et Archétype (MODE COMPLET, après couches 1-3)

**Pour détecter l'Ombre :**

| Question | Ce qu'elle révèle |
|---|---|
| "Qu'est-ce qui vous irrite profondément chez certaines personnes — sans pouvoir expliquer pourquoi ?" | Projection probable |
| "Y a-t-il une façon d'être que vous admirez de façon irrationnelle mais pensez ne jamais pouvoir avoir ?" | Ombre dorée |
| "Quel aspect de vous-même vous semble le plus difficile à admettre ?" | Accès direct à l'Ombre |
| "Quand vous pensez à la personne que vous ne voulez surtout pas devenir — qui est-elle ?" | Ombre par l'opposé |

**Pour identifier l'archétype :**

| Question | Signal |
|---|---|
| "Comment vous comportez-vous habituellement face à une difficulté — vous avancez, vous vous retirez, vous aidez, vous contrôlez ?" | Pattern comportemental de l'archétype |
| "De quoi avez-vous le plus peur dans votre vie en ce moment ?" | Peur de l'archétype rigidifié |
| "Y a-t-il quelque chose dans cette situation que vous portez seul — sans avoir demandé d'aide ?" | Signal Héros / Protecteur / Orphelin |

---

### COUCHE 5 — Flexibilité ACT

| Question | Ce qu'elle révèle |
|---|---|
| "Cette pensée que vous avez — est-ce une pensée ou est-ce vrai ?" | Niveau de fusion |
| "Si cette histoire que vous vous racontez n'était pas vraie — que changeriez-vous ?" | Distance possible du récit |
| "Ce que vous feriez si vous n'aviez pas peur — c'est quoi ?" | Valeurs sous la peur |
| "Qu'est-ce qui compte vraiment pour vous, même si ça coûte quelque chose ?" | Valeurs propres vs d'emprunt |
| "Y a-t-il quelque chose que vous évitez depuis longtemps — et que vous savez qu'il faudrait faire ?" | Évitement comportemental |

## Séquences recommandées selon l'entrée initiale

### Entrée : Répétition
1. (Couche 1) "Cette conviction — depuis quand est-elle là ?"
2. (Couche 2) "La partie de vous qui freine — qu'est-ce qu'elle essaie d'éviter ?"
3. (Couche 3) "Si votre histoire avait un titre — quel serait-il ?"
4. (Couche 5) "Si cette histoire que vous vous racontez n'était pas vraie — que changeriez-vous ?"
5. (Couche 2) "Quand cette émotion monte — à quel âge vous sentiez-vous déjà comme ça ?"

### Entrée : Conflit interne
1. (Couche 2) "La partie de vous qui freine — qu'est-ce qu'elle essaie d'éviter ?"
2. (Couche 2) "Si cette résistance avait une intention positive cachée — laquelle ?"
3. (Couche 1) "Cette conviction — depuis quand est-elle là ?"
4. (Couche 5) "Cette pensée que vous avez — est-ce une pensée ou est-ce vrai ?"
5. (Couche 3) "Y a-t-il quelqu'un dont la voix résonne quand vous vous critiquez ?"

### Entrée : Limite invisible
1. (Couche 1) "Qu'est-ce qui se passerait de pire si vous alliez plus loin ?"
2. (Couche 5) "Ce que vous feriez si vous n'aviez pas peur — c'est quoi ?"
3. (Couche 1) "Qui attendez-vous que vous soyez dans cette situation ?"
4. (Couche 2) "Y a-t-il une partie de vous qui peut observer tout ça avec distance ?"
5. (Couche 3) "Quel tournant dans votre histoire aurait pu tout changer ?"
